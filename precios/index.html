Scripts y Datos

Primero vas a necesitar los siguientes scripts, pónlos en cualquier carpeta del sistema que tiene la plataforma de precios:

  • usuarios.csv
  • pathConfig.php
  • generatePassword.php
  • newUsers.php
  • registerUsers.php

El .csv sólo es una muestra, llena este archivo con los datos de tus usuarios.

Pasos a Seguir

  1. Edita pathConfig.php para que la carpeta $precios_path apunte a la carpeta raíz de la plataforma de precios.
  2. El archivo usuarios.csv debe de tener los datos de los usuarios en formato "nombre,apellido paterno, apellido materno, email."
  3. Corre la instrucción php newUsers.php > usuarios_pass.csv, esto generará contraseñas para los usuarios y te avisará si estos ya estaban registrados en el sistema de precios, elimina los usuarios ya registrados de usuarios_pass.csv (busca las palabras MAIL o USER).
  4. Corre el último script php registerUsers.php para registrar a los usarios en la base de datos.
