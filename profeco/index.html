Primero corres el programa de escrapeo de profeco,

$ sbt
> run

Copia los scripts auxiliares a la carpeta de datos y entra a la carpeta de datos,

$ cp src/main/resources/* data/
$ cd data/

Corre el script de preparación de datos. Una vez finalizado, entra a la base MySQL donde quieres guardar tus datos,

$ perl sql_prepare.pl
$ mysql --host=metronhomo.com.mx --user=metrooya_user metrooya_profeco_todo -p

Desde aquí corre los scripts sql, uno escribe los registros a la base total de profeco, el otro filtra resultados a la base filtrada de profeco a productos de interés de Dante,

> source sql_fill.sql;
> source sql_filter.sql;

Tendrás que editar el último script para reflejar el mes de levantamientos que quieres sincronizar (month=número).
